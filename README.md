# SlippingAbove

A 2d platforming foddian game like getting over it.

## Controls

Use left mouse button to hold and drag in the opposite direction of in which you want to go. Upon releasing left mouse button after dragging, you will jump in the opposite direction of mouse drag.

## Characters

1. You
2. Happy
3. Happy's Brother
4. Happy's Other Brother
.. rest is just Happy's more brothers whom you meet along the way.

## Tools for creation
Unity 6
Adobe Photoshop 2025
